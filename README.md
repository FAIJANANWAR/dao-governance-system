# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/FAIJANANWAR/dao-governance-system.git
```

Navigate to project:

```bash
cd dao-governance-system/artifacts/dao-governance
```

Install dependencies:

```bash
pnpm install
```

Run development server:

```bash
pnpm dev
```
